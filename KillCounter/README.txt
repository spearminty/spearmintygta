KILLSTREAK KILL COUNTER


When you kill a pedestrian or destroy a vehicle a tally of your kills will appear. After 15 seconds if you have not earned a kill the counter will reset.


PREREQUISITES:

- Microsoft .NET Framework 4.5
- Microsoft Visual C++ Redistributable Package
- Script Hook V and the ASI loader
- Script Hook V Dot Net


INSTALLATION:

- If you have the old .dll version of this mod installed, remove it.
- Place the KillCounter.dll file into the "scripts" folder in your main GTAV directory. Create the folder if it doesn't exist.
